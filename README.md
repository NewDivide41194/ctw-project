# G123 React Assignment

## Technology Stack
- This project primarly used [ReactJS], [ES6] + [TypeScript].
- Used [Redux_toolkit] to manage states.
- Styled components by using [Tailwind_CSS].

## Setup tool
>Vite


## Dependencies
- "@reduxjs/toolkit": "^1.9.2",
- "react": "^18.2.0",
- "react-dom": "^18.2.0",
- "react-redux": "^8.0.5"

## Installation and Run
```bash
npm install
npm run dev
```
